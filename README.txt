Il presente elaborato si divide in 4 sotto-progetti: ogni progetto riprodurrà uno dei 4 grafici rappresentati nell'articolo commissionato nel file "Testo.pdf". 

In particolare:
- La cartella Project4A contiene tutti i file per l'esecuzione di un programma Java che cercherà di emulare il risultato in figura 4A dell'articolo.
- La cartella Project4B contiene tutti i file per l'esecuzione di un programma Java che cercherà di emulare il risultato in figura 4B dell'articolo.
- La cartella Project6A contiene tutti i file per l'esecuzione di un programma Java che cercherà di emulare il risultato in figura 6A dell'articolo.
- La cartella Project6B contiene tutti i file per l'esecuzione di un programma Java che cercherà di emulare il risultato in figura 6B dell'articolo.

Per lanciare i programmi, bisogna:
1) Avviare il proprio ambiente di sviluppo Java.
2) Aprire di volta in volta ogni cartella Project.
3) Accertarsi che la propria JDK sia impostata sulla versione 12.0 o superiore.
4) Si deve dire al proprio IDE di includere i file "weka.jar" e "xchart-3.6.1.jar" nella sezione dipendenze di progetto. I suddetti file sono contenuti, per ogni progetto, nella cartella "lib". Questi file serviranno a fornire il software necessario per l'implementazione di classificatori e grafici.
5) Build and run.
